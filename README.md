# Unity Clicker

A small clicker game created in **Unity** as a personal project.
The game was developed in just **two days** as a fun gift for my girlfriend.

## About the project

This is a simple clicker-style game where the player earns points by clicking and progresses through basic upgrades.
The project was created mainly for practice and as a small, personal surprise.

## Features

* Simple click-based gameplay
* Basic upgrade system
* Clean and minimal interface
* Created in Unity using C#

## Development

* Engine: Unity
* Language: C#
* Development time: 2 days
* Purpose: Personal/learning project

Created as a small personal project and gift.
